# Blinkit-sales-dashboard-PowerBI
Sales dashboard

## 📊 Project Title

BlinkIT Grocery Sales & Outlet Performance Dashboard

## 🚀 Short Description / Purpose

This project showcases an interactive dashboard built using Microsoft Power BI to analyze BlinkIT’s grocery sales data.
It provides deep insights into sales trends, outlet performance, product categories, and customer preferences, enabling data-driven decision-making for retail optimization.

## 🛠️ Tech Stack
- Microsoft Power BI – Dashboard creation & data visualization
- Microsoft Excel – Data source & preprocessing
- Data Analysis – Data cleaning, transformation & insight generation

  ## 📊 Key Visuals & Insights
🔸 **1. Outlet Establishment Trend**
Shows sales trend over years
Peak observed around 2018 (~₹205K)
Insight: Certain years/outlets significantly outperform → useful for expansion strategy
🔸 **2. Sales by Fat Content**
Comparison between Low Fat vs Regular products
Regular products contribute higher sales (~₹776K vs ₹425K)
Insight: Customer preference leans toward regular items → inventory planning opportunity
🔸 **3. Item Type Performance**
Top categories:
  🥇 Fruits & Snacks (~₹0.18M each)
  🥈 Household & Frozen Foods
Lowest: Seafood, Breakfast
Insight: Focus marketing & stocking on high-performing categories
🔸 **4. Outlet Size Distribution**
Medium outlets dominate sales (~₹507K)
Followed by Small and High-size outlets
Insight: Medium-sized outlets are most efficient
🔸 **5. Outlet Location Analysis**
Tier 3 cities generate highest sales (~₹472K)
Tier 2 and Tier 1 follow
Insight: Strong demand in Tier 3 → high growth potential
🔸 **6. Outlet Type Comparison**
Supermarket Type1 leads significantly (~₹787K sales, 5577 items)
Grocery stores contribute less comparatively
Insight: Supermarket model is most scalable and profitable
🔸 **7. Fat Content by Outlet**
Tier-wise comparison of low-fat vs regular items
Regular products dominate across all tiers
Insight: Consistent customer preference pattern

## 💡Business Impact
- Identifies high-performing outlets and locations
- Enables better inventory and category planning
- Supports data-driven expansion strategies
- Improves understanding of customer buying behavior
